typedef bool SOPC_SetTargetVariables_Func(OpcUa_WriteValue *nodesToWrite, int32_t nbValues)

The subscriber scheduler calls this callback cyclically to pass the values received by the subscriber.

Note
Ownership of the WriteValue array and its elements is transferred to the callback code which must free them.
This function is called once per configured DataSet, with all the values of the DataSet in a single call.
Returns
true if processed, false otherwise (array must still be freed)
