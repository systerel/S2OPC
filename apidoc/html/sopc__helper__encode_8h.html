Get the number of padding characters for base64 ('='). Starting from the end until the first base64 characters.

Parameters
inputA valid pointer to the input.
outLenthe number of padding characters (0, 1 or 2)
Returns
SOPC_STATUS_OK when successful otherwise SOPC_STATUS_INVALID_PARAMETERS if input is NULL or has not a number of padding characters equal to 0, 1 or 2.
