SOPC_AddressSpace * SOPC_Embedded_AddressSpace_LoadWithAlloc ( bool  allocNodes)

Load the embedded address space defined as a C structure. If the C structure is defined as a const address space, allocNodes should be set to false and will be ignored.

Parameters
allocNodestrue if the nodes shall be allocated and copied to fill the address space, false otherwise.
Returns
The loaded address space in case of success, NULL otherwise.
Note
allocNodes shall be set to true to allocate and copy the nodes, it makes then possible to use AddNode and DeleteNode services when S2OPC_NODE_MANAGEMENT is defined.
Warning
If the C structure is defined as a const address space, allocNodes will be ignored.

SOPC_AddressSpace * SOPC_Embedded_AddressSpace_Load ( void  )

Load the embedded address space defined as a C structure. The address space resulting uses the nodes of the provided structure without copying them even when those are not const. As a result AddNode and DeleteNode services will not be available.

Deprecated:
This function is deprecated and will be removed in future releases. Use SOPC_Embedded_AddressSpace_LoadWithAlloc(false) instead for same behavior.
Returns
The loaded address space in case of success, NULL otherwise.
