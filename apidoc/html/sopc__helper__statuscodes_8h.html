Convert a status code to its string representation (allocating memory)

Parameters

Returns
An allocated string containing the name of the status code constant, or the hexadecimal representation if the status code is not recognized. The caller is responsible for freeing the returned string with SOPC_Free(). Returns NULL if memory allocation fails.
