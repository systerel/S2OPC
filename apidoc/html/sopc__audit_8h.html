typedef enum SOPC_Audit_OptionMask SOPC_Audit_OptionMask

Provides the interface for user-configuration Auditing. Only non-auditing application can be defined if S2OPC_HAS_AUDITING is not defined.

Configuration options for Auditing. Any combination of following values is possible.

Note
For now, only security-related options are supported (see SOPC_Audit_SupportedSecuOptions).
