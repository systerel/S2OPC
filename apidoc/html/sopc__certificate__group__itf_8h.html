SOPC_CertificateGroup_Config **  ppConfig 
)

Get the CertificateGroup object configuration with the default address space.

Parameters
groupTypeDefine the certificate group type of the TrustList (application or user).
certTypeDefine the certificate type (certificate properties).
pPKIA valid pointer to the TrustList PKI that belongs to the CertificateGroup object.
maxTrustListSizeDefine the maximum size in byte of the TrustList that belongs to the CertificateGroup object.
pKeyCertPairA valid pointer to the private key and certificate that belongs to the CertificateGroup object (NULL if certType is SOPC_TRUSTLIST_GROUP_USR ).
pKeyPathPath to the private key that belongs to the CertificateGroup object (NULL if certType is SOPC_TRUSTLIST_GROUP_USR or if the platform has no file system).
pCertPathPath to the certificate that belongs to the CertificateGroup object (NULL if certType is SOPC_TRUSTLIST_GROUP_USR or if the platform has no file system).
[out]ppConfigA newly created configuration. You should delete it with SOPC_CertificateGroup_DeleteConfiguration .
Returns
SOPC_STATUS_OK if successful.
