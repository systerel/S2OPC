◆ TrustList_Read()

SOPC_ReturnStatus TrustList_Read ( SOPC_TrustListContext *  pTrustList,
int32_t  reqLength,
SOPC_ByteString *  pDest 
)

Read reqLength bytes from the current position of the encoded TrustListDataType. If reqLength is greater than the encoded data then the whole TrustListDataType is read otherwise the position is incremented by reqLength . The trustList part read (trusted certs/CRL, issuers certs/CRL) depends on the OpeningMask.

Parameters
pTrustListThe TrustList context.
reqLengthThe length byte to read (!= 0)
[out]pDestA valid byte string to store the result.
Warning
pTrustList shall be valid (!= NULL) and opened in read mode.
Returns
SOPC_STATUS_OK if successful.

bool *  pbIsRemove,
bool *  pbIsIssuer 
)

Remove a single Certificate from the TrustList. If the Certificate is a CA Certificate that has CRLs then all CRLs for that CA are removed as well.

Parameters
pTrustListThe TrustList context.
pThumbprintThe thumbprint of the certificate to remove formatted as a hexadecimal string.
bIsTrustedCertIf TRUE the certificate is removed from the trusted certificates list otherwise the certificate is removed from the issuer certificates list.
[out]pbIsRemoveA valid pointer indicating whether the certificate has been found and deleted.
[out]pbIsIssuerA valid pointer indicating whether the deleted certificate is an issuer.
Warning
pTrustList shall be valid (!= NULL) and not opened.
Returns
SOPC_GoodGenericStatus if successful.
