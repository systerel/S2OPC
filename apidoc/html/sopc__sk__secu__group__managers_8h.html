Detailed Description

SK security group managers stores the SOPC_SKManager to use to retrieve the keys for different security groups.

The initialization SOPC_SK_SecurityGroup_Managers_Init shall be called prior to any other function of this module and the clear SOPC_SK_SecurityGroup_Managers_Clear when this module need to be reset or is not used anymore.

To define a security group manager, SOPC_SK_SecurityGroup_Managers_Init and SOPC_SK_SecurityGroup_SetSkManager shall be called.

The Publisher and Subscriber schedulers will then automatically call SOPC_PubSubSKS_GetSecurityKeys.
