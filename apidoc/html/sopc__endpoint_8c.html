Open the endpoint and listen for connections from clients (TCP listener), then wait for TCP UA connection (TCP UA Hello reception and TCP UA Ack response) and secure channel establishement (TCP UA OpenSecureChannel request reception and send TCP UA OpenSecureChannel response) Note: in single threaded mode, it is necessary to call the socket manager loop to receive messages.

Parameters
endpointThe channel to connect
urlEndpoint address of the connection point
callbackEndpoint events callback function to be called
callbackDataData to be provided to the endpoint event callback function on call
serverCertificateServer certificate to use for responding to clients connections (or NULL for None security mode)
serverKeyServer private key to use for responding to clients connections (or NULL for None security mode)
pkiThe Public Key Infrastructure to use for validating certificates (or NULL for None security mode)
nbSecuConfigsNumber of security policy supported and presents in secuConfigurations
secuConfigurationsArray of security policies supported with nbSecuConfigs length
Returns
STATUS_OK if endpoint opening succeeded, STATUS_NOK otherwise
