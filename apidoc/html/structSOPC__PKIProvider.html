const SOPC_FnValidateCertificate SOPC_PKIProvider::pFnValidateCertificate

The validation function, which is wrapped by CryptoProvider_Certificate_Validate().

It implements the validation of the certificate. The CryptoProvider_Certificate_Validate() assumes that a STATUS_OK from this function means that the certificate can be trusted. Parameters are validated by CryptoProvider_Certificate_Validate().

Parameters
pPKIA valid pointer to the PKIProvider.
pToValidateA valid pointer to the Certificate to validate.
Returns
STATUS_OK when the certificate is successfully validated, and STATUS_INVALID_PARAMETERS or STATUS_NOK.
