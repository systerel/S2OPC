Register action to start the channel connection establishment (TCP connection, TCP UA Hello message) and the secure channel establishement (TCP UA OpenSecureChannel request) on server acknowledgment (TCP UA Ack message).

Parameters
channelThe channel to connect
urlEndpoint address for establishing the connection
crt_cliClient certificate to use for establishing the connection (or NULL for None security mode)
key_privClient private key to use for establishing the connection (or NULL for None security mode)
crt_srvServer certificate of the endpoint server to connect (or NULL for None security mode)
pkiThe Public Key Infrastructure to use for validating certificates (or NULL for None security mode)
reqSecuPolicyUriURI of the requested security policy
requestedLifetimeLifetime requested for the channel connection
msgSecurityModeSecurity mode to use for the messages exchanged through the connection
networkTimeoutNetwork timeout to establish the connection (in milliseconds)
eventCbConnection state changed callback function to be called (connection established, disconnection)
eventCbDataData to be provided to the connection state changed callback function on call
connectCbAsynchronous connect result callback function called on connection failure or success
connectCbDataAsynchronous connect result callback data provided to callback function on call
Returns
STATUS_OK if channel connection start step succeeded, STATUS_NOK otherwise
